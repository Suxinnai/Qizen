# 栖知 信息架构（IA）

> 版本 v0.1 · 2026-04-17 · 作者：灵 & 沐灵
> 这份文档定义栖知所有页面、它们的关系、用户如何流转

---

## 一、整体地图

```
栖知（macOS / Windows 桌面应用）
│
├─ 🚪 [应用首启] 学习风格测试（onboarding）
│   └─ 一次性，5-8 分钟
│   └─ 完成后跳转 → 看板
│
├─ 🏠 看板（Dashboard）              ← 默认首页
├─ 📚 学习空间（Study）⭐            ← 核心工作区
├─ 🎯 我的目标（Goals）
├─ 🗂 资料库（Library）
├─ 🌳 知识图谱（Graph）
├─ 📝 笔记（Notes）
└─ ⚙️ 设置（Settings）
    └─ 含「我的学习画像」可视化与重测入口

[全局浮动]
└─ 💫 「灵」浮动入口（每个主页面右下角）
   └─ 点击 → 唤起对话浮窗 / 跳转学习空间
```

---

## 二、页面定位与职责

| 页面 | 类型 | 核心职责 | 用户停留时长 |
|---|---|---|---|
| Onboarding | 一次性流程 | 初始化用户画像 | 5-8 分钟（仅一次） |
| 看板 | 总览页 | 一眼看全今天该做什么 | 1-3 分钟 |
| **学习空间** ⭐ | **工作区** | **真正学习的地方** | **20-60 分钟** |
| 我的目标 | 管理页 | 看长目标全景、调整计划 | 5-10 分钟 |
| 资料库 | 管理页 | 上传资料、做练习 | 10-30 分钟 |
| 知识图谱 | 探索页 | 看自己的知识网络 | 5-15 分钟 |
| 笔记 | 沉淀页 | 复习、整理 | 10-30 分钟 |
| 设置 | 配置页 | 调偏好、看画像 | 1-5 分钟 |

## 三、各页面的入口与出口

### 3.1 学习风格测试（Onboarding）

**入口**：
- 应用首次启动（自动）
- 设置 → 重新测试（手动）

**出口**：
- 完成 → 看板
- 跳过（不允许，必须完成才能进主界面）

**画像更新**：
- 后续每次学习会话后，后台静默更新（无需用户操作）

---

### 3.2 看板（Dashboard）

**入口**：
- 默认首页
- 侧栏「看板」
- 任何页面 → Cmd+1

**出口**：
- 点击今日任务 → **学习空间**（自动定位到该任务）
- 点击复习提醒 → **学习空间**（复习模式）
- 点击知识图谱缩略 → **知识图谱**
- 点击「灵」浮窗 → **学习空间**（自由对话模式）

**关键转化点**：
- 看板 → 学习空间 是最高频的跳转，必须**一键到位**

---

### 3.3 学习空间（Study）⭐ 核心

**入口**：
- 看板今日任务卡片
- 看板复习提醒
- 「灵」浮窗
- 知识图谱节点 → "去学习"
- 我的目标 → 任务卡 → "开始学习"
- 笔记 → "回到当时的对话"

**出口**：
- 学习完毕 → 自动弹出"本次会话总结"卡 → 看板
- 点击右栏「当前位置」节点 → 知识图谱
- 点击 AI 找到的资源 → 系统浏览器打开外链
- 番茄结束 → 系统通知 + 暂停

**子状态**：
- 教学模式（默认）：AI 主导讲解
- 复习模式：从看板复习提醒进入，AI 直接出题
- 自由对话模式：从「灵」浮窗进入，无固定话题

---

### 3.4 我的目标（Goals）

**入口**：
- 侧栏「我的目标」
- 看板今日任务点击"查看完整计划"
- Cmd+3

**出口**：
- 点击任务 → **学习空间**
- 新建目标 → 弹出"和灵聊聊你想学什么" → AI 拆解后跳到该目标详情

---

### 3.5 资料库（Library）

**入口**：
- 侧栏「资料库」
- 拖拽文件到桌面应用窗口（自动打开资料库）
- 看板没有直接入口（资料库是独立工作流）

**出口**：
- 点击资料 → 详情页（查看 / 重新出题 / 关联到目标）
- 开始练习 → **学习空间**（练习模式）
- 资料关联到知识点 → **知识图谱**

---

### 3.6 知识图谱（Graph）

**入口**：
- 侧栏「知识图谱」
- 看板缩略图 → "展开完整图谱"
- 学习空间右栏「当前位置」点击放大

**出口**：
- 点击节点 → 详情面板（不离开此页）
- 详情面板 → "去学习" → **学习空间**
- 详情面板 → "查看相关笔记" → **笔记**

---

### 3.7 笔记（Notes）

**入口**：
- 侧栏「笔记」
- 学习空间右栏临时笔记 → "查看完整笔记"
- 知识图谱节点 → "查看相关笔记"

**出口**：
- 点击笔记 → 在中栏打开
- "回到当时的对话" → **学习空间**（载入历史会话）
- 导出 → 系统文件保存对话框

---

### 3.8 设置（Settings）

**入口**：
- 侧栏「设置」
- 顶栏用户头像菜单
- Cmd+,

**子模块**：
- **我的学习画像**（最重要）：
  - 雷达图可视化
  - "上次更新：3 天前"
  - "重新测试" → **Onboarding**
  - 历史变化曲线（画像如何随时间变化）
- 学习偏好
- AI 教学偏好
- 通知设置
- 数据导出
- 关于栖知

## 四、关键用户旅程

### 旅程 1：新用户首次使用

```
应用启动
  ↓
Onboarding 欢迎页
  ↓
做 36 道题（5-8 分钟）
  ↓
看见自己的「学习风格画像」+ 灵的解读
  ↓
点击「开始我的学习之旅」
  ↓
进入【看板】
  ↓
看到引导："和灵聊聊，告诉它你想学什么"
  ↓
点击「灵」浮窗
  ↓
进入【学习空间·自由对话】
  ↓
和灵聊：「我想三个月学完高数上」
  ↓
灵自动拆解 → 创建目标 → 跳到该目标详情
  ↓
回到【看板】，看到今日任务已生成
```

### 旅程 2：日常学习

```
打开应用
  ↓
【看板】看到今日还剩 2 个任务
  ↓
点击「高数·微分中值定理」任务
  ↓
进入【学习空间】，自动加载该知识点
  ↓
左栏：高亮当前位置在计划树
中栏：灵开始苏格拉底式提问
右栏：番茄钟自动开始 25 分钟
  ↓
学完一个小节，点「✅ 我懂了」
  ↓
灵更新画像（后台静默）
  ↓
番茄结束 → 系统通知 → 休息 5 分钟
  ↓
继续下一个知识点 / 关闭
  ↓
回到【看板】，看到今日已完成 4/5
```

### 旅程 3：复习

```
打开应用
  ↓
【看板】看到「⏰ 该复习了：链式法则」
  ↓
点击「开始复习 5 分钟」
  ↓
进入【学习空间·复习模式】
  ↓
灵直接出 3 道题
  ↓
答错 → 灵分析错因 + 用新方式讲一遍
  ↓
答对 → 更新该节点的掌握度
  ↓
完成 → 回看板
```

### 旅程 4：基于资料学习

```
拖拽 PDF 到应用窗口
  ↓
自动打开【资料库】
  ↓
显示"灵正在学习..."进度条
  ↓
完成 → 提取出 12 个知识点 → 让用户确认
  ↓
确认后弹出："要不要把这份资料关联到「高数」目标？"
  ↓
确认 → 资料库出现新卡片
  ↓
点击"开始练习"
  ↓
进入【学习空间·练习模式】
  ↓
灵基于资料出题 + 实时讲解
```

## 五、全局浮动元素

### 「灵」浮动入口

**出现位置**：每个主页面（看板/目标/资料/图谱/笔记/设置）右下角
**不出现**：学习空间（已经在和灵对话了）、Onboarding

**默认形态**：圆形按钮，「灵」SVG 头像，呼吸动画
**Hover**：展开为提示卡 "想聊聊吗？"
**点击**：
- 在看板：跳到学习空间·自由对话模式
- 在其他页：弹出迷你对话浮窗（不离开当前页）

**全局快捷键**：Cmd+Space 唤起灵浮窗（macOS）

## 六、导航规则

### 6.1 侧栏菜单（始终存在）

```
🏠 看板          Cmd+1
📚 学习空间       Cmd+2
🎯 我的目标       Cmd+3
🗂 资料库        Cmd+4
🌳 知识图谱       Cmd+5
📝 笔记          Cmd+6
⚙️ 设置          Cmd+,
```

### 6.2 面包屑（部分页面）

- 学习空间：`高数上 > 第3章 > 拉格朗日中值定理`
- 资料库 / 笔记 / 图谱：根据当前选中项动态显示

### 6.3 返回逻辑

- 学习空间：右上角 ✕ 关闭 → 回上一个页面（默认看板）
- 详情面板：Esc 关闭

## 七、空状态设计

| 页面 | 空状态文案 | 空状态行动 |
|---|---|---|
| 看板（无任务） | "今天还没目标呢，要不和灵聊聊？" | 唤起灵 |
| 学习空间（无活跃会话） | "想学什么？告诉灵" | 输入框 focus |
| 我的目标 | "还没有目标。学习从一个清晰的方向开始" | "新建目标" |
| 资料库 | "把你的课件、笔记、PDF 拖进来吧" | 上传按钮 |
| 知识图谱 | "学的越多，这片森林越茂盛" | 跳转学习空间 |
| 笔记 | "学习时记下灵感，灵会帮你整理" | 跳转学习空间 |

**所有空状态**：必须有一片手绘叶子 SVG + 一句温和文案 + 一个明确行动按钮

## 八、错误状态

| 类型 | 处理 |
|---|---|
| 网络错误（AI 调用失败） | 重试按钮 + "灵暂时连不上网" |
| API 限流 | "稍等一下下，灵在喘口气" |
| 文件解析失败 | "这个文件灵看不懂，要不换一个？" |
| 数据库错误 | 顶栏黄色横幅 + "数据保存可能有问题，建议重启" |

## 十、IA 验收标准

每个页面要回答 3 个问题：

1. **从哪里来？** 至少 2 个明确入口
2. **在这里做什么？** 一个核心动作
3. **下一步去哪？** 至少 1 个明确出口

避免"信息孤岛"：每个页面都要能流转，不能让用户走进一个死胡同。
