# 栖知 Qizhi

> 栖息于知识之中 —— 一个真正"懂你"的 AI 私人学习导师

## 项目定位

**用 AI 一对一的能力，打破传统"一个老师教一个班"的固定教学模式，给每个学习者一个真正适配自己学习风格、节奏、情绪的私人导师。**

## 核心差异化

不是题库，不是网课，不是又一个换皮 ChatGPT。
栖知的核心是 **"懂你"**：
- 懂你怎么学得进去（学习风格画像）
- 懂你今天的状态（情绪感知）
- 懂你学到哪了（知识图谱）
- 懂你什么时候该复习（遗忘曲线）

## 当前阶段

**作品打磨期** —— 先把核心体验做透，验证"AI 私教"模式真的有效，再考虑商业化。

## 形态

**macOS / Windows 桌面应用**（Tauri 2.0 + React + SQLite）

不做 Web 版本。理由：
- 学习是沉浸式的，桌面应用专注度更高
- 本地数据，隐私优先（用户画像、笔记、资料都存本地）
- 可接入系统级能力（托盘、快捷键、通知、文件拖拽）
- 视觉表现力更强（毛玻璃、原生菜单）

## 目标用户（首发）

**大学生**

