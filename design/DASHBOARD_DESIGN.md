# 栖知 看板设计方向

> 版本 v0.2 · macOS 桌面 App · Notion 极简 + 栖知 DNA
> 更新：明确 macOS 桌面应用形态、加入窗口框架与毛玻璃侧栏

---

## 一、设计理念

### 核心三原则

1. **少即是多** —— 留白是美学，也是认知负担的减法
2. **数据有温度** —— 不是冷冰冰的图表，是"我看到自己在变强"
3. **AI 隐于其中** —— 不抢戏，但随时唤起

### 视觉调性参考

- **Notion 桌面版**（结构和留白）
- **Linear**（动效与质感）
- **Things 3**（任务呈现 + macOS 原生感）
- **Craft**（东方雅致 + 桌面 App 优雅）
- **Apple Fitness**（数据可视化的温度）

## 二、桌面应用窗口框架（重要！）

栖知是 **macOS / Windows 桌面应用**，每个页面都套在一个统一的应用窗口里。

```

├──────────┬──────────────────────────────────────────────────┤
│          │                                                  │
│ 毛玻璃    │                                                  │
│ 侧栏      │             主内容区                             │
│ 220px    │                                                  │
│          │                                                  │
│ 🏠 看板  │                                                  │
│ 📚 学习  │                                                  │
│ 🎯 目标  │                                                  │
│ 🗂 资料  │                                                  │
│ 🌳 图谱  │                                                  │
│ 📝 笔记  │                                                  │
│ ⚙️ 设置  │                                                  │
│          │                                                  │
│ ─────    │                                                  │
│ 👤 沐灵  │                                                  │
└──────────┴──────────────────────────────────────────────────┘
```

### 标题栏（36px）
- macOS：左侧红黄绿三圆点（系统级，但栖知 Tauri 自定义实现）
- 中间：当前页面标题（如"栖知 — 看板"）
- 右侧：暗黑模式切换 + 通知铃铛
- 背景：rgba(255,255,255,0.6) + backdrop-filter blur(20px)
- 整个标题栏可拖动窗口（`data-tauri-drag-region`）

### 侧栏（220px · 毛玻璃）
- 背景：rgba(250,248,244,0.6) + backdrop-filter blur(40px) saturate(1.8)
- 顶部：栖知 Logo（叶子 SVG + "栖知"思源宋体）
- 菜单 7 项（Lucide 图标 16px + 文字 13px）
- 当前页：栖知青背景 12% + 文字栖知青 + 左侧 2px 高亮条
- 菜单 hover：rgba(0,0,0,0.04)
- 底部：用户头像 + "沐灵" + 在线状态点

### 主内容区
- 背景：纯白（暗模式 #1E1E1C）
- 内边距：32px
- 卡片之间：gap-6（24px）
- 滚动条：macOS 风（细、半透明、hover 才出现）

## 四、模块详细设计

### 4.1 顶部问候卡（横跨整行）

```
┌──────────────────────────────────────────────────┐
│                                                  │
│   ☀️ 早上好，沐灵                  🍃            │
│                                                  │
│   连续学习 23 天 🔥  ·  今天还有 2 个任务         │
│   ────────────────────────────────────────       │
│   "见微知著，学有所栖"                            │
│                                                                 🍃 │
└──────────────────────────────────────────────────┘
```

**细节**：
- 高度 180px
- 背景：linear-gradient(135deg, #F5F2EC, #E8F0EC) + 极淡 SVG 噪点纹理
- 大字"早上好，沐灵"：Cormorant Garamond + Noto Serif SC，36px，#2D7A6B
- 副行：14px，灰色 #8A8478
- 格言：思源宋体斜体，13px，灰色
- **手绘叶子 SVG**：右上角和右下角散落 2-3 片（栖知青，opacity 30%）
- 暗模式：渐变换成深色版（#1E1E1C → #2A2A28）

**根据时间切换问候语**：
- 5-11 时："☀️ 早上好"
- 11-13 时："🌤 中午好"
- 13-18 时："🍃 下午好"
- 18-23 时："🌙 晚上好"
- 23-5 时："🌌 还在学呀，记得早点休息"

**格言文案库**（不重复）：
- "见微知著，学有所栖"
- "千里之行，始于足下"
- "今日所学，明日所栖"
- "知不足，然后能自反"
- "学而时习之，不亦说乎"

## 五、交互细节

### 5.1 微动效原则
- **进入动画**：fade in + slight slide up（300ms, ease-out）
- **数字变化**：number flip 动画
- **任务完成**：描边划过 + 轻微缩放
- **页面切换**：crossfade 250ms
- 尊重 macOS"减少动态效果"系统设置

### 5.2 反馈
- 所有操作 < 100ms 视觉响应
- 加载用骨架屏，不用转圈圈
- 错误用气泡，不用弹窗
- 系统级通知（番茄钟结束、复习提醒）

### 5.3 空状态
- 不写"暂无数据"
- 写"还没开始呀，先和灵聊聊想学什么？"

## 六、暗黑模式

必须支持，且要做到**深色比浅色更美**。
- 跟随 macOS 系统主题
- 用户也可在标题栏手动切换
- 参考：Linear、Vercel Dashboard

**关键暗模式适配**：
- 窗口外层：#0E0E0D
- 窗口本体：#1E1E1C
- 卡片：#2A2A28
- 标题栏：rgba(0,0,0,0.3) + blur
- 侧栏：rgba(20,20,18,0.6) + blur
- 问候卡渐变：#1E1E1C → #2A2A28
- 文字：#DDD8CE（正文）/ #8A8478（次要）
